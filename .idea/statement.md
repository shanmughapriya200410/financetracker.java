# Project Statement

## Problem Statement
Managing personal finances effectively remains a challenge for many users due to lack of straightforward tools. This project aims to build a console-based Personal Finance Tracker to help users record, manage, and analyze their incomes and expenses easily.

## Project Scope
The scope includes user management, categorized transaction handling, summary reporting, and data export functionality. The project focuses on implementing core Java programming skills and good software design practices.

## Target Users
- Students and young professionals seeking simple finance management
- Individuals interested in tracking spending habits and goals

## High-Level Features
- User registration and login system
- Add/Edit/Delete income and expense transactions
- Categorized transactions for better financial insights
- Generate and view financial summaries
- Export transaction data to CSV for external use
- Input validation and basic error handling
